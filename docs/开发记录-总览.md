# Marina —— 开发记录

> 一款 Windows 终端管理器，目标对位 Windows Terminal / Tabby / Wave / Warp。

## 一句话摘要

**4 天 12 小时**自然时间内（其中**真正在敲键盘的小时数约 19 个**），从空仓库写出一个可打包、可装、可跑、跟 Windows Terminal 同赛道、并在它没做的地方（会话不死、托盘常驻、多窗口共享会话池、Win11 新右键菜单集成）领先的桌面应用，并完成 alpha.3 发布前的两轮勘误。

## 功能清单（V1 alpha 实际交付）

- 路径管理（书签 / 活跃 / 最近，三栏侧边栏）
- 会话生命周期 `active/idle/exited`，**关窗不杀会话**，托盘常驻
- 启动模板：Shell / Claude Code / Codex / OpenCode 内置 + 自定义
- 多窗口共享会话池，跨窗口可见
- OSC 1337 CWD 跟踪（PowerShell + cmd.exe），CWD 漂移 ⚠ 提示
- 7 主题（Rose Pine 系列 + Cutie / Business / Ubuntu / Windows Terminal）
- macOS / Windows 双标题栏样式
- 设置：实时应用、无保存按钮、配置导入导出（含敏感字段擦除）
- Explorer 经典右键 + **Win11 新右键菜单**（IExplorerCommand）
- 终端搜索（Ctrl+F）、bracketed paste、多行粘贴 confirm、ANSI 注入警告
- WebGL 渲染器 + DOM 回退
- 自绘 Modal/Confirm/Prompt（替代 `window.confirm/prompt`，因为 Electron 不支持）
- 持久化日志、render-process-gone 自愈、ConPTY resize、shell 探测缓存

## 体系化决策（非随手写）

- **ADR-008**：路径与 CWD 解耦，砍掉"路径墓地"机制
- **IPC 协议文档化**（`docs/ipc-protocol.md`），typed 收发
- **平台抽象**（`src/main/platform/`），macOS/Linux 接口已留好
- **终端渲染审计备忘录** §1–11，把 CPB-P/C、FOC、TYP、IPC、CRA、OSC、PER、SEC、FLK、RSZ、XTM、STM、CUR 12 个错误类前缀全部分类编号，每个 fix 都引这个编号
- 测试覆盖：persistence 18、PathManager 36、SettingsManager 31、SessionManager 21；启动 + 交互双层冒烟脚本
